### Identifying and Converting Data Types
The ```type()``` function verifies the data type of a datum.
```python
print(type("String"))
print(type(1))
print(type(1.0))
print(type(["a", "b", "c"]))
print(type(("a", "b", "c")))
print(type({"a", "b", "c"}))
print(type({"a":1, "b":2, "c":3}))
```
> <class 'str'>\
> <class 'int'>\
> <class 'float'>\
> <class 'list'>\
> <class 'tuple'>\
> <class 'set'>\
> <class 'dict'>
