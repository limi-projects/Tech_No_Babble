Tech, No Babble

This weblog aims to clarify various technical concepts across the fields of chemistry, physics, mathematics, computing, electronics and intellectual property law. Throughout, I have tried to adhere to Orwell's essay: Politics and the English Language, which outlines guidance for clear and accessible writing. Additionally, a Technobabble section may be provided at the end of some pages, to concisely clarify certain terms (Yes, I will occasionally split an infinitve - please don't call the Grammar Police!).

Page Design

This weblog is written in HTML, CSS and JavaScript, and is hosted via GitHub Pages. Source code for further languages may be provided. However, only JavaScript code runs within the page. Page function and clarity is prioritised over page aesthetics; so apologies for my questionable choice of colours, formats and fonts!

The content of this weblog is accessed and navigated via the the upper naviagtion bar.
